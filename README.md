# SuperSave 💰

Projeto final do curso de Análise e Desenvolvimento de Sistemas (2023).

### Objetivo 🎯

O aplicativo, feito com tecnologias como Ionic e Firebase, tem como objetivo ajudar o indivíduo a fazer suas contas do mês para melhor planejamento financeiro.
